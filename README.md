<b>Приложение</b>

<b>Запуск приложения</b><br/>
В корневой папке выполните:

<pre><code>docker compose up</code></pre>

Приложение будет собрано и запущено в контейнере.

<b>Запуск тестов</b><br/>
Для запуска тестов используйте:

<pre><code>pytest</code></pre>

Все тесты находятся в папке <code>tests/</code>.

<pre><code>Alembic</code></pre>

Все миграции находятся в папке <code>migrations/</code>.
